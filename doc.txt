# Документация к проекту 2048

## Общее описание
2048 - это простая и увлекательная игра, в которой игроки объединяют плитки с одинаковыми числами, чтобы достичь плитки с номером 2048. Игра имеет динамическое обновление счета, баланса и таблицы лидеров.

## Структура проекта
1. **index.html** - Главная страница игры.
2. **victory.html** - Страница таблицы лидеров.
3. **rules.html** - Страница с правилами игры.
4. **rules.txt** - Файл с текстом правил игры.
5. **game.js** - Логика игры, реализованная с использованием классов.
6. **controls.js** - Управляющий скрипт для обработки событий.
7. **style.css** - Стили для игры, включая адаптивный дизайн и анимации.
8. **doc.txt** - Документация проекта.

## Установка и запуск
1. Скачайте проект на локальный компьютер.
2. Откройте файл `index.html` в любом веб-браузере.
3. Играйте, следуя правилам, перечисленным на странице `rules.html`.

## Как играть
- Перемещайте плитки влево, вправо, вверх и вниз, чтобы объединять одинаковые числа.
- Нажмите на кнопки управления для выполнения действий, таких как "вернуться на один ход", "удалить плитку" и "перемешать плитки".
- После окончания игры введите свое имя и сохраните результат в таблице лидеров.
